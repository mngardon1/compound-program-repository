Install Your Strategist Coach

Your AI partner for the entire 90 days. Installed once, gains skills as you progress. The conductor of every conversation, every decision, every artifact you produce.

20 min install · Pro plan required (Claude or ChatGPT)
What You're Installing

One coach. Three skills. From day one.

You're installing the Strategist Coach with three skills already loaded into its knowledge base. The coach won't have any business context yet — that's expected. The skills are the procedures the coach will run as you produce that context over the next 90 days.

Skill 01 · Phase 1.1

Compound Diagnostic

30-min baseline conversation across Person, Problem, POV. Produces your first canonical file. The first thing you'll invoke after install.

Skill 02 · Phase 1.2

Positioning Architect

60–90 min positioning build. Produces 7 library files (narrative-spine, frameworks, core-problem, pain-maps, desire-maps, icp, positioning). Iterates against market signal.

Skill 03 · Phase 2

Offer Architect

Guided Offer Doc build using load-bearing hierarchy, value equation stress test, structured guarantee questions. Loaded now; used in Phase 2.

More skills get added in later phases as their work begins — sales page creator, pipeline diagnosis, pricing decision, content strategy. The Strategist Coach is the constant; the skills are how it grows with you.

You install one coach today. It conducts the entire program — Phase 1 positioning, Phase 2 offer, Phase 3 pipeline, and everything that follows.
Before You Install

Two things must be ready.

Both are non-negotiable. If either is missing, fix it first.

1. A paid AI subscription

Pick one. You do not need both.

  • Claude Pro ($20/mo) — recommended. The skill system runs more reliably in Claude. Free Claude works for testing but file limits will frustrate you.
  • ChatGPT Plus ($20/mo) — also works. ChatGPT executes skill files less faithfully than Claude (about 70–80% adherence vs ~95%). The Strategist will still help; expect slightly looser enforcement of the failure-mode checks.

2. Your Compound Consultant O.S. folder

If you skipped this in 0.1: create a folder on your computer called /Compound Consultant OS. Anywhere you'll actually use it — Dropbox, Google Drive, Desktop, iCloud. Your canonical files will live here as you produce them.

No Phase 1 files yet — that's expected

You haven't produced any canonical files yet (the diagnostic in Phase 1.1 will be your first). Your knowledge library is empty on Day 1. That's fine. The Strategist Coach knows it's installed before any positioning work exists and will guide you to invoke the Diagnostic skill in your first real session.

Option A — Install on Claude

Six steps. About fifteen minutes.

  1. Create a new Claude Project
    Go to claude.ai/projects. Click + New Project. Name it Strategist Coach. Description can be blank.
  2. Add custom instructions (the system prompt)
    Click Set custom instructions. Copy File 1 above and paste it in. Save.
  3. Save the three skill files to your computer
    Copy each of Files 2, 3, and 4 above. Paste each into a plain text file. Save with the filenames shown: skill-compound-diagnostic.md, skill-positioning-architect.md, skill-offer-architect.md.
  4. Upload all three skill files to the project's knowledge base
    In your Strategist Coach project, click Add content. Upload all three skill files. The Strategist reads these to know which procedures it can invoke.
  5. Start a chat — introduce yourself
    Click Start chat in project. Type "Hi, I just installed you. I'm ready to start the program." The Strategist will recognize there are no canonical files yet and will guide you to Phase 1.1 (the Diagnostic skill).
  6. Bookmark the project
    This is your daily AI workspace for the entire program. Pin it. Add to dock. You'll return here more than any other tool.
If Claude says it can't see the skill files

Confirm the files appear in the Knowledge area. Re-upload as .md or .txt (not Word docs). Start a new chat and explicitly say: "Read the skill files in your knowledge base before we begin."

Option B — Install on ChatGPT

Six steps. About fifteen minutes.

  1. Create a new Custom GPT
    Go to chatgpt.com/gpts/editor. Click Configure.
  2. Fill the basics
    Name: Strategist Coach. Description: one-liner. Leave conversation starters blank.
  3. Paste the system prompt into Instructions
    Copy File 1 above. Paste into the Instructions field.
  4. Save the three skill files locally + upload to Knowledge
    Save Files 2, 3, 4 as .md files on your computer. In the GPT builder, scroll to Knowledge, click Upload files, upload all three.
  5. Save as private, start chatting
    Click Create → Only me. Click View GPT. Type "Hi, I just installed you. I'm ready to start the program."
  6. Pin it in your ChatGPT sidebar
    This is your daily workspace. Make it the first thing you see.
ChatGPT file limits

Custom GPTs cap knowledge at 20 files. Your initial install uses 3. Plenty of room for canonical files and additional skills as the program continues.

How You'll Use It

Every session starts in the same place.

You open the Strategist Coach project. You type what you want to work on — or just say hi. The Strategist reads your knowledge files, identifies what you should focus on, and either:

  • Answers strategically in chat (pricing question, "what should I work on next," diagnose a stuck funnel)
  • Invokes a skill for structured production work (e.g., "let's run the diagnostic" triggers the Diagnostic skill)
  • Scopes a project brief for work that belongs in a separate project (content production, automation, IP development)

The first session after install will almost always be the Diagnostic skill — that's Phase 1.1. After that, the coach has your baseline and starts producing meaningful output every time.

The Strategist isn't a faster ChatGPT. It's a strategic partner that remembers your business across sessions and gets sharper every week you use it.
